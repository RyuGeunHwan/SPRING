# 에러종류

## 404 오류 대표적인 예

```
1. 없는 url입력
2. url 오타
3. @RestController OR @Controller 어노테이션 입력X
```

## 405 오류 대표적인 예

```
1. get인데 post데이터 전송(HTTP 메소드 불일치)
```

## 500(백엔드에서 오류) 오류 대표적인 예

```
1. 자바 코딩을 잘못했을 때
 ex) 배열길이가 10인데, 11번째 인덱스를 호출
     쿼리 오타
     쿼리 결과 List인데 List로 안받을 때
```

## 403 오류 대표적인 예

```
1. 권한 에러
```
