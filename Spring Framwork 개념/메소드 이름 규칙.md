# method name

```
1. Controller : 메소드 이름 앞에 call or load 많이 사용
    ex)
    callEmpInfo() ~
    loadEmpInfo() ~

2. Service : 메소드 이름 앞에 get, set,  동사로 표현~(make, upload, download 등등...)

3. Mapper :
    메소드 이름 앞에
    1. (JPA를 사용할 경우 많이 사용) find(select), save(insert, update), delete
    2. get, set,
    3. 쿼리이름(select, update, insert, delete)
```
